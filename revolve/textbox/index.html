You look at the Project Re:VOLVE textbox generator.

There’s no way to download your custom textbox yet, so what I do is hit the print screen key, paste it into GIMP, use the rectangle select tool to select only the textbox, select Image → Crop to Selection, select Image → Crop to Content, use the Select by Color Tool to select one of the borders (make sure the threshold is set to 0%), then hit the delete key to erase all of them, making the textbox transparent. You’ll have to save it to maintain transparency, as copying it to your clipboard will make it black again. If you plan on uploading it somewhere, you may want to zoom in a bit first to fight back against JPG compression.

The font was manually redrawn using BitFontMaker 2. You can download the font here, but keep in mind it may be updated in the future. Aside from ^ it has the same characters as the original bitmap font. This means that on mobile, you may have to press and hold your quotation mark keys until the straight ' " quotation marks appear, as smart quotes are not included.

Enjoy!
